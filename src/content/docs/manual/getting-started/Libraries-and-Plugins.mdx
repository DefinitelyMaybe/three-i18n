Listed here are some externally developed compatible libraries and plugins for three.js.
[Threejsresources](https://threejsresources.com/) is also a useful place to find three.js related content

### Physics

<ul>
<li>[ Oimo.js](https://github.com/lo-th/Oimo.js/)</li>
<li>[ enable3d](https://enable3d.io/)</li>
<li>[ ammo.js](https://github.com/kripken/ammo.js/)</li>
<li>[ cannon-es](https://github.com/pmndrs/cannon-es)</li>
<li>[ rapier](https://rapier.rs/)</li>
<li>[ Jolt](https://github.com/jrouwe/JoltPhysics.js)</li>

</ul>

support for some additional effects and frameworks are available through external libraries.

<ul>
  <li>[ postprocessing](https://github.com/vanruesc/postprocessing)</li>
</ul>

### Wrappers and Frameworks

<ul>
  <li>[ A-Frame](https://aframe.io/)</li>
  <li>
    [ Lume](https://lume.io/) - HTML elements for 3D graphics built on Three.
  </li>
  <li>
    [ react-three-fiber](https://github.com/pmndrs/react-three-fiber) - React
    components for 3D graphics built on Three.
  </li>
  <li>
    [ threepipe](https://threepipe.org/) - A versatile 3D viewer framework using
    three.js for rendering.
  </li>
  <li>[ ECSY](https://github.com/ecsyjs/ecsy-three)</li>
  <li>
    [ Threlte](https://threlte.xyz/) - Svelte components for 3D graphics built

  </li>
  <li>[ Needle Engine](https://needle.tools/)</li>
  <li>
    [ tresjs](https://tresjs.org/) - Vue components for 3D graphics built on
    Three.
  </li>
  <li>
    [ Giro3D](https://giro3d.org) - Versatile framework built on Three for
    visualizing and interacting with Geospatial 2D, 2.5D and 3D data.
  </li>
  <li>
    [ Mattercraft](https://zap.works/mattercraft/) - Browser-based visual editor
    for AR, WebXR and 3D web content, built on three.js with real-time preview
    and physics engine.
  </li>
</ul>
