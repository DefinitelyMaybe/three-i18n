## 1. DOM + CSS

<div>

使用HTML通常是最简单、最快速的添加文本的方法，这是大多数的Three.js示例中用于添加描述性叠加文字的方法。

你可以在这里添加内容

```javascript
<div id="info">Description</div>
```

然后使用CSS来将其绝对定位在其它具有z-index的元素之上，尤其是当你全屏运行three.js的时候。

```javascript

#info {
position: absolute;
top: 10px;
width: 100%;
text-align: center;
z-index: 100;
display:block;
}

```

</div>

## 4. three.js自带的文字几何体

<div>

如果你更喜欢使用纯three.js来工作，或者创建能够由程序改变的、动态的3D文字，你可以创建一个其几何体为THREE.TextGeometry的实例的网格：

```javascript
new THREE.TextGeometry(text, parameters);
```

然而，为了使得它能够工作，你的TextGeometry需要在其“font”参数上设置一个THREE.Font的实例。

请参阅 [TextGeometry](examples/geometries/TextGeometry) 页面来阅读如何完成此操作的详细信息，以及每一个接收的参数的描述，还有由three.js分发、自带的JSON字体的列表。

## 5. 位图字体

<div>

BMFonts (位图字体) 可以将字形批处理为单个BufferGeometry。BMFont的渲染支持自动换行、字母间距、字句调整、signed distance fields with standard derivatives、multi-channel signed distance fields、多纹理字体等特性。
详情请参阅 [link:https://github.com/felixmariotto/three-mesh-ui three-mesh-ui](http://www.jaanga.com/2012/03/blender-to-threejs-create-3d-text-with.html]) 或 [ three-bmfont-text](https://github.com/Jam3/three-bmfont-text)。

现有库存的字体在项目中同样可用，就像[ A-Frame Fonts](https://github.com/etiennepinchon/aframe-fonts)一样，
或者你也可以从任何TTF字体中创建你自己的字体，优化时，只需要包含项目中所需的字符即可。

这是一些有用的工具：

<ul>
<li>[ msdf-bmfont-web](http://msdf-bmfont.donmccurdy.com/) _(web-based)_</li>
<li>[ msdf-bmfont-xml](https://github.com/soimy/msdf-bmfont-xml) _(commandline)_</li>
<li>[ hiero](https://github.com/libgdx/libgdx/wiki/Hiero) _(desktop app)_</li>
</ul>
</div>
