يقدم هذا الدليل لمحة موجزة عن المكونات الأساسية لتطبيق VR مخصص للويب بإستعمال three.js.

## سير العمل

أولاً ، عليك ضم [ VRButton.js](https://github.com/mrdoob/three.js/blob/master/examples/jsm/webxr/VRButton.js) في مشروعك.

```javascript
import { VRButton } from "three/addons/webxr/VRButton.js";
```

_VRButton.createButton()_ يقوم بأمرين مهمين: أولا يقوم بإنشاء زر يشير إلى توافق الواقع الافتراضي. إلى جانب ذلك ، يبدأ جلسة VR إذا قام المستخدم بتنشيط الزر. كل ما عليك فعله هو إضافة السطر التالي من التعليمات البرمجية إلى تطبيقك.

```javascript
document.body.appendChild(VRButton.createButton(renderer));
```

بعد ذلك ، عليك توجيه نموذج _WebGLRenderer_ لتمكين عرض XR.

```javascript
renderer.xr.enabled = true;
```

أخيرًا ، يجب عليك ضبط حلقة الرسوم المتحركة لأننا لا نستطيع استخدام وظيفة _window.requestAnimationFrame()_ المعروفة لدينا. بالنسبة لمشاريع الواقع الافتراضي ، نستخدم [setAnimationLoop](api/renderers/WebGLRenderer#setAnimationLoop).
يبدو الحد الأدنى من الكود كما يلي:

```javascript
renderer.setAnimationLoop(function () {
  renderer.render(scene, camera);
});
```
