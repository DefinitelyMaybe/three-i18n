Digamos que você queira desenhar uma linha ou um círculo, não um wireframe [Mesh](api/objects/Mesh).
Primeiro precisamos configurar o [renderizador](api/renderers/WebGLRenderer) (renderer), a [cena](api/scenes/Scene) (scene) e
a câmera (camera) (veja a página Criando uma cena).

Aqui está o código que vamos usar:

```javascript
const renderer = new THREE.WebGLRenderer();
renderer.setSize(window.innerWidth, window.innerHeight);
document.body.appendChild(renderer.domElement);

const camera = new THREE.PerspectiveCamera(
  45,
  window.innerWidth / window.innerHeight,
  1,
  500,
);
camera.position.set(0, 0, 100);
camera.lookAt(0, 0, 0);

const scene = new THREE.Scene();
```

A próxima coisa que vamos fazer é definir um material.
Para linhas nós temos que usar [LineBasicMaterial](api/materials/LineBasicMaterial) ou [LineDashedMaterial](api/materials/LineDashedMaterial).

```javascript
//create a blue LineBasicMaterial
const material = new THREE.LineBasicMaterial({ color: 0x0000ff });
```

Depois do material, nós vamos precisar de uma geometria com alguns vértices:

```javascript
const points = [];
points.push(new THREE.Vector3(-10, 0, 0));
points.push(new THREE.Vector3(0, 10, 0));
points.push(new THREE.Vector3(10, 0, 0));

const geometry = new THREE.BufferGeometry().setFromPoints(points);
```

Note que linhas são desenhadas entre cada par consecutivo de vértices,
mas não entre o primeiro e o último (a linha não é fechada).

Agora que nós temos os pontos para duas linhas e um material,
podemos juntar tudo e formar uma linha

```javascript
const line = new THREE.Line(geometry, material);
```

Tudo o que falta é adicioná-la na cena e chamar o [renderizador](api/renderers/WebGLRenderer#render).

```javascript
scene.add(line);
renderer.render(scene, camera);
```

Agora você deve estar vendo uma seta apontando para cima, feita de duas linhas azuis.
