Defined in: three/addons/postprocessing/SSAOPass.js:43

A pass for a basic SSAO effect.

[SAOPass](SAOPass.html) and GTAPass produce a more advanced AO but are also
more expensive.

```js
const ssaoPass = new SSAOPass( scene, camera, width, height );
composer.addPass( ssaoPass );
```
