Defined in: three/addons/interactive/SelectionBox.js:41

This class can be used to select 3D objects in a scene with a selection box.
It is recommended to visualize the selected area with the help of [SelectionHelper](SelectionHelper.html).

```js
const selectionBox = new SelectionBox( camera, scene );
const selectedObjects = selectionBox.select( startPoint, endPoint );
```
