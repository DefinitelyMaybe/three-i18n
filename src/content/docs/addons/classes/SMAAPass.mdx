Defined in: three/addons/postprocessing/SMAAPass.js:24

A pass for applying SMAA. Unlike FXAAPass, `SMAAPass` operates in
`linar-srgb` so this pass must be executed before [OutputPass](OutputPass.html).

```js
const smaaPass = new SMAAPass();
composer.addPass( smaaPass );
```
