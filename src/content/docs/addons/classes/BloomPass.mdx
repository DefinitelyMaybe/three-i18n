Defined in: three/addons/postprocessing/BloomPass.js:25

A pass for a basic Bloom effect.

[UnrealBloomPass](UnrealBloomPass.html) produces a more advanced Bloom but is also
more expensive.

```js
const effectBloom = new BloomPass( 0.75 );
composer.addPass( effectBloom );
```
