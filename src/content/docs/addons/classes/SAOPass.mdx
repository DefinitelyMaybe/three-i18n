A SAO implementation inspired from

## Bhouston

previous SAO work.

`SAOPass` provides better quality than [SSAOPass](SSAOPass.html) but is also more expensive.

```js
const saoPass = new SAOPass( scene, camera );
composer.addPass( saoPass );
```
