A loader for the PVRTC texture compression format.

```js
const loader = new PVRLoader();

const map = loader.load( 'textures/compressed/disturb_4bpp_rgb.pvr' );
map.colorSpace = THREE.SRGBColorSpace; // only for color textures
```
