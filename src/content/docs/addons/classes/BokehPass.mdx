Defined in: three/addons/postprocessing/BokehPass.js:29

Pass for creating depth of field (DOF) effect.

```js
const bokehPass = new BokehPass( scene, camera, {
	focus: 500
	aperture: 5,
	maxblur: 0.01
} );
composer.addPass( bokehPass );
```
