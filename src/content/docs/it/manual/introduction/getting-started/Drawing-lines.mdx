Diciamo che tu voglia disegnare una linea o un cerchio, non un wireframe [Mesh](api/objects/Mesh).
Prima bisogna impostare il [renderer](api/renderers/WebGLRenderer), la [scene](api/scenes/Scene) e la camera (vedi la pagina Creare una scena).

Di seguito il codice che utilizzeremo:

```javascript
const renderer = new THREE.WebGLRenderer();
renderer.setSize(window.innerWidth, window.innerHeight);
document.body.appendChild(renderer.domElement);

const camera = new THREE.PerspectiveCamera(
  45,
  window.innerWidth / window.innerHeight,
  1,
  500,
);
camera.position.set(0, 0, 100);
camera.lookAt(0, 0, 0);

const scene = new THREE.Scene();
```

La prossima cosa da fare è definire il materiale. Per le linee possiamo usare [LineBasicMaterial](api/materials/LineBasicMaterial) o [LineDashedMaterial](api/materials/LineDashedMaterial).

```javascript
//crea una LineBasicMaterial blu
const material = new THREE.LineBasicMaterial({ color: 0x0000ff });
```

Dopo la definizione del materiale abbiamo bisogno di una geometria con alcuni vertici:

```javascript
const points = [];
points.push(new THREE.Vector3(-10, 0, 0));
points.push(new THREE.Vector3(0, 10, 0));
points.push(new THREE.Vector3(10, 0, 0));

const geometry = new THREE.BufferGeometry().setFromPoints(points);
```

Da notare che le linee vengono tracciate tra ogni coppia consecutiva di vertici, ma non tra il primo e l'ultimo vertice (la linea non è chiusa).

Ora che abbiamo i punti per due linee e un materiale, possiamo unirli per formare una linea.

```javascript
const line = new THREE.Line(geometry, material);
```

Non resta che aggiungerlo alla scena e chiamare il [render](api/renderers/WebGLRenderer#render).

```javascript
scene.add(line);
renderer.render(scene, camera);
```

Ora dovresti vedere una freccia che punta verso l'alto formata da due linee blu.
