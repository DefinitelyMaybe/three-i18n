## Caricamento

Solo alcuni loader (ad es. [ObjectLoader](api/loaders/ObjectLoader)) sono inclusi di default con three.js ‚Äî altri devono essere aggiunti all'applicazione individualmente.

```javascript
import { GLTFLoader } from "three/addons/loaders/GLTFLoader.js";
```

Una volta importato un loader, sei pronto per aggiungere e caricare un modello alla scena. La sintassi
varia a seconda del loader utilizzato - quando vengono utilizzati altri formati, contralla gli esempi e la
documentazione del loader. Per il formato glTF, l'utilizzo con gli script globali sarebbe:

```javascript
const loader = new GLTFLoader();

loader.load(
  "path/to/model.glb",
  function (gltf) {
    scene.add(gltf.scene);
  },
  undefined,
  function (error) {
    console.error(error);
  },
);
```

Per maggiori dettagli consulta la [documentazione GLTFLoader](examples/loaders/GLTFLoader).
