このガイドでは、three.jsを使って作成したWebベースのVRアプリケーションの基本的なコンポーネントの概要を説明しています。

## Workflow

はじめに、プロジェクトに[ VRButton.js](https://github.com/mrdoob/three.js/blob/master/examples/jsm/webxr/VRButton.js)を追加する必要があります。

```javascript
import { VRButton } from "three/addons/webxr/VRButton.js";
```

_VRButton.createButton()_ は2つの重要なことを行います。VR互換性であることを表すボタンを作成し、ユーザーがボタンをアクティブにするとVRセッションを開始します。この機能を追加するためには、次のコードをアプリに追加するだけで良いです。

```javascript
document.body.appendChild(VRButton.createButton(renderer));
```

次に _WebGLRenderer_ インスタンスのXRレンダリングを有効にしてください。

```javascript
renderer.xr.enabled = true;
```

最後にアニメーションループを調整する必要があります。というのも、よく知られた _window.requestAnimationFrame()_ 関数を使えないからです。VRプロジェクトのためには、[setAnimationLoop](api/renderers/WebGLRenderer#setAnimationLoop)を使います。最低限必要なコードは以下のようになります。

```javascript
renderer.setAnimationLoop(function () {
  renderer.render(scene, camera);
});
```
