이 가이드에서는 three.js를 통한 웹 기반 VR 앱의 기본 컴포넌트를 만드는 방법을 알려드리겠습니다.

## 작업 순서

먼저, [ VRButton.js](https://github.com/mrdoob/three.js/blob/master/examples/jsm/webxr/VRButton.js)
를 프로젝트에 불러옵니다.

```javascript
import { VRButton } from "three/addons/webxr/VRButton.js";
```

*VRButton.createButton()*은 두 가지 중요한 일을 합니다: VR에서 활용이 가능한 버튼을 만듭니다. 그리고 유저가 버튼을 누르면
VR 세션을 실행시킵니다. 다음 코드를 삽입하기만 하면 됩니다.

```javascript
document.body.appendChild(VRButton.createButton(renderer));
```

다음으로, _WebGLRenderer_ 인스턴스에게 XR 렌더링을 허용해줘야합니다.

```javascript
renderer.xr.enabled = true;
```

마지막으로, 자주 쓰이는 _window.requestAnimationFrame()_ 기능을 활용할 수 없기 때문에, 애니메이션 루프를 수정해주어야 합니다.
VR 프로젝트에서는 [setAnimationLoop](api/renderers/WebGLRenderer#setAnimationLoop)를 사용합니다.
가장 간소화된 코드는 다음과 같습니다:

```javascript
renderer.setAnimationLoop(function () {
  renderer.render(scene, camera);
});
```
