This class enables element access on instances of [StorageBufferNode](/reference/threewebgpu/classes/storagebuffernode/).
In most cases, it is indirectly used when accessing elements with the
[StorageBufferNode#element](/reference/threewebgpu/classes/storagebuffernode/#element) method.

```js
const position = positionStorage.element( instanceIndex );
```
