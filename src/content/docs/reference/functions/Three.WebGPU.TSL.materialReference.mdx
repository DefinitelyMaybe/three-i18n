> **materialReference**(`name`, `type`, `material`?): [`MaterialReferenceNode`](../classes/Three.WebGPU.MaterialReferenceNode.html)

Defined in: three/src/nodes/accessors/MaterialReferenceNode.js:85

TSL function for creating a material reference node.

## Parameters

### name

`string`

The name of the property the node refers to.

### type

`string`

The uniform type that should be used to represent the property value.

### material?

`Material` = `null`

The material the property belongs to.
When no material is set, the node refers to the material of the current rendered object.

[`MaterialReferenceNode`](../classes/Three.WebGPU.MaterialReferenceNode.html)
