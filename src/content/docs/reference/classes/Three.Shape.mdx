Defined in: three/src/extras/core/Shape.js:35

Defines an arbitrary 2d shape plane using paths with optional holes. It
can be used with [ExtrudeGeometry](Three.ExtrudeGeometry.html), [ShapeGeometry](Three.ShapeGeometry.html), to get
points, or to get triangulated faces.

```js
const heartShape = new THREE.Shape();

heartShape.moveTo( 25, 25 );
heartShape.bezierCurveTo( 25, 25, 20, 0, 0, 0 );
heartShape.bezierCurveTo( - 30, 0, - 30, 35, - 30, 35 );
heartShape.bezierCurveTo( - 30, 55, - 10, 77, 25, 95 );
heartShape.bezierCurveTo( 60, 77, 80, 55, 80, 35 );
heartShape.bezierCurveTo( 80, 35, 80, 0, 50, 0 );
heartShape.bezierCurveTo( 35, 0, 25, 25, 25, 25 );

const extrudeSettings = {
	depth: 8,
	bevelEnabled: true,
	bevelSegments: 2,
	steps: 2,
	bevelSize: 1,
	bevelThickness: 1
};

const geometry = new THREE.ExtrudeGeometry( heartShape, extrudeSettings );
const mesh = new THREE.Mesh( geometry, new THREE.MeshBasicMaterial() );
```

### arcLengthDivisions

> **arcLengthDivisions**: `number`

Defined in: three/src/extras/core/Curve.js:37

This value determines the amount of divisions when calculating the
cumulative segment lengths of a curve via [Curve#getLengths](Three.Curve.html#getlengths). To ensure
precision when using methods like [Curve#getSpacedPoints](Three.Curve.html#getspacedpoints), it is
recommended to increase the value of this property if the curve is very large.

#### Default
