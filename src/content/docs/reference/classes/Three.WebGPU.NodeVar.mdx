Defined in: three/src/nodes/core/NodeVar.js:7

[NodeBuilder](Three.WebGPU.NodeBuilder.html) is going to create instances of this class during the build process
of nodes. They represent the final shader variables that are going to be generated
by the builder. A dictionary of node variables is maintained in [NodeBuilder#vars](Three.WebGPU.NodeBuilder.html#vars) for
this purpose.
