Defined in: three/src/nodes/core/NodeAttribute.js:7

[NodeBuilder](Three.WebGPU.NodeBuilder.html) is going to create instances of this class during the build process
of nodes. They represent the final shader attributes that are going to be generated
by the builder. Arrays of node attributes is maintained in [NodeBuilder#attributes](Three.WebGPU.NodeBuilder.html#attributes)
and [NodeBuilder#bufferAttributes](Three.WebGPU.NodeBuilder.html#bufferattributes) for this purpose.
