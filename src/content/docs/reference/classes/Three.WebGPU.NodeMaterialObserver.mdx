Defined in: three/src/materials/nodes/manager/NodeMaterialObserver.js:59

This class is used by [WebGPURenderer](Three.WebGPU.WebGPURenderer.html) as management component.
It's primary purpose is to determine whether render objects require a
refresh right before they are going to be rendered or not.
