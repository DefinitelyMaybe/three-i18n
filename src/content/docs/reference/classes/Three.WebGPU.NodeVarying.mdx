Defined in: three/src/nodes/core/NodeVarying.js:11

[NodeBuilder](Three.WebGPU.NodeBuilder.html) is going to create instances of this class during the build process
of nodes. They represent the final shader varyings that are going to be generated
by the builder. An array of node varyings is maintained in [NodeBuilder#varyings](Three.WebGPU.NodeBuilder.html#varyings) for
this purpose.
