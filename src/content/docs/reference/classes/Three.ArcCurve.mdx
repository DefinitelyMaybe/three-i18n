### arcLengthDivisions

> **arcLengthDivisions**: `number`

Defined in: three/src/extras/core/Curve.js:37

cumulative segment lengths of a curve via [Curve#getLengths](Three.Curve.html#getlengths). To ensure
precision when using methods like [Curve#getSpacedPoints](Three.Curve.html#getspacedpoints), it is
recommended to increase the value of this property if the curve is very large.
