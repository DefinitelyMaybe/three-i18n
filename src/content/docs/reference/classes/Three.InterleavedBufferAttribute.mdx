Defined in: three/src/core/InterleavedBufferAttribute.js:12

An alternative version of a buffer attribute with interleaved data. Interleaved
attributes share a common interleaved data storage ([InterleavedBuffer](Three.InterleavedBuffer.html)) and refer with
different offsets into the buffer.
