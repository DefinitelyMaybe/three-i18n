Defined in: three/src/nodes/core/NodeCode.js:7

[NodeBuilder](Three.WebGPU.NodeBuilder.html) is going to create instances of this class during the build process
of nodes. They represent user-defined, native shader code portions that are going to be
injected by the builder. A dictionary of node codes is maintained in [NodeBuilder#codes](Three.WebGPU.NodeBuilder.html#codes)
for this purpose.
