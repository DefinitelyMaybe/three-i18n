Defined in: three/src/nodes/core/NodeFrame.js:9

Management class for updating nodes. The module tracks metrics like
the elapsed time, delta time, the render and frame ID to correctly
call the node update methods [Node#updateBefore](Three.WebGPU.Node.html#updatebefore), [Node#update](Three.WebGPU.Node.html#update)
and [Node#updateAfter](Three.WebGPU.Node.html#updateafter) depending on the node's configuration.
