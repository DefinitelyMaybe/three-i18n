Defined in: three/src/core/UniformsGroup.js:15

A class for managing multiple uniforms in a single group. The renderer will process
such a definition as a single UBO.

Since this class can only be used in context of [ShaderMaterial](Three.ShaderMaterial.html), it is only supported
in [WebGLRenderer](Three.WebGLRenderer.html).
