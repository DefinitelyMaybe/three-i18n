Defined in: three/src/nodes/core/NodeUniform.js:7

[NodeBuilder](Three.WebGPU.NodeBuilder.html) is going to create instances of this class during the build process
of nodes. They represent the final shader uniforms that are going to be generated
by the builder. A dictionary of node uniforms is maintained in [NodeBuilder#uniforms](Three.WebGPU.NodeBuilder.html#uniforms)
for this purpose.
