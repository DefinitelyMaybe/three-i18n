## Properties

### arcLengthDivisions

> **arcLengthDivisions**: `number`

Defined in: three/src/extras/core/Curve.js:37

This value determines the amount of divisions when calculating the
cumulative segment lengths of a curve via [Curve#getLengths](#getlengths). To ensure
precision when using methods like [Curve#getSpacedPoints](#getspacedpoints), it is
recommended to increase the value of this property if the curve is very large.

#### Default

```ts
200
```
