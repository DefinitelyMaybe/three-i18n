Defined in: three/src/nodes/lighting/AONode.js:10

A generic class that can be used by nodes which contribute
ambient occlusion to the scene. E.g. an ambient occlusion map
node can be used as input for this module. Used in [NodeMaterial](Three.WebGPU.NodeMaterial.html).
